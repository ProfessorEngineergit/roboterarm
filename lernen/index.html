1 Was ist ein Terminal?

Zwei Wege, dem Computer etwas zu sagen

Du kennst den ersten Weg schon: klicken mit der Maus. Der zweite ist das Terminal — du tippst den Befehl direkt. Schneller, mächtiger, und Computerprofis lieben es.

🖱️ Mit der Maus (GUI)

Du siehst Bildchen und klickst. Einfach — aber langsam, wenn du viele Dinge machen willst.

⌨️ Mit dem Terminal (CLI)

Terminal
du$ ls ordner/ datei.txt du$ öffne datei.txt ▌

Du tippst, was passieren soll. Blitzschnell, und du kannst sogar ganze Abläufe automatisieren. Das lernst du jetzt. 🚀

2 Terminal öffnen

So startest du es

Auf jedem Computer gibt es ein Terminal. So findest du es:

🪟 Windows · Weg 1

Drücke Windows-Taste + R, tippe cmd, Enter.

🪟 Windows · Weg 2

Klicke Start, tippe Terminal und öffne die App.

🤖 Am Roboter (wir!)

Wir steuern den Roboter-Computer aus der Ferne — mit ssh.

Terminal
du$ ssh radxa@roboterarm-01.local Passwort: ▌ ✓ Willkommen am Roboterarm!

💡 Auf dem Mac: Drücke Cmd + Leertaste, tippe Terminal, Enter. Genau dasselbe.

4 Roboterarm-Spickzettel

Alle Befehle für den Roboter

Wenn du am Roboter-Computer bist (per ssh), brauchst du genau diese Befehle. Klick auf „Kopieren" und füg sie ins Terminal ein.

① Projekt holen & einrichten

# Projekt vom Server holen git clone https://github.com/ProfessorEngineergit/roboterarm ~/roboterarm cd ~/roboterarm # Alles installieren (Pakete + Servos) ./install.sh

② Server starten

# Den Roboter-Webserver starten cd ~/roboterarm python3 service/robot_service.py # Dann im Browser öffnen: # http://10.42.0.1:8765

③ Neueste Version laden

# Updates holen + neu starten cd ~/roboterarm git pull sudo systemctl restart roboterarm

④ Kalibrieren

# Servos einstellen (oder im Web-Tab ⚙) cd ~/roboterarm python3 calibrate.py
